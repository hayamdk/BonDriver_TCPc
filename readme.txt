・説明
mod_tcpcastへ接続してストリームを受信するBonDriver

・使い方
BonDriver_TCPc.dllとBonDriver_TCPc.ChSet.txt（新規作成）をTVTest等のディレクトに設置。
ChSet.txtの書式例は以下のとおり。
（tsdump.exeを192.168.0.2で実行していて、以下のようなポートを指定している場合）

----------------------
$地上波
192.168.0.2:1234
192.168.0.2:1235
192.168.0.2:1236
192.168.0.2:1237
$BS
192.168.0.2:1238
192.168.0.2:1239
$CS
192.168.0.2:1240
192.168.0.2:1241
----------------------
